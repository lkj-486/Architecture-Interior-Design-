# 景观设计
- ## 为建筑增添绿植
  在场景布置的时候，绿植的布置工作量是很大的，而且由于大部分建筑同学没有相对应的景观知识，容易选择错误的绿植。

# 建筑渲染
- ## 多角度渲染时，不改变建筑的结构与立面材质
  现有的Stable Diffusion在多角度渲染时，容易出现建筑立面材质不统一、结构不统一等问题，需要更精确的控制。
